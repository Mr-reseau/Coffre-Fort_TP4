# 🔒 Coffre-Fort Numérique

Un gestionnaire de secrets sécurisé développé en Python pour le TP de cryptographie.

## 🎯 Objectif du TP

Implémentation d'un coffre-fort numérique avec :
* Authentification par mot de passe haché (SHA-256)
* Chiffrement symétrique AES-256 CBC
* Gestion sécurisée des clés
